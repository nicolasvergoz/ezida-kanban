## 1. Board helpers + typed errors (`internal/board/columns.go`)

- [ ] 1.1 Create `internal/board/columns.go`. Declare the four new typed errors: `EmptyColumnNameError` (no fields), `ColumnAlreadyExistsError{Name string}`, `CannotDeleteLastColumnError{Name string}`, and `ColumnHasCardsError{Name string, Cards []affectedCard}`. Each MUST have an `Error() string` method. Also declare an internal `affectedCard` struct with `ID string \`json:"id"\`` and `Title string \`json:"title"\``. Reuse the existing `*ColumnNotFoundError` from `board.go`. Done when `go vet ./internal/board` is clean and the new types appear in `go doc ./internal/board`.
- [ ] 1.2 Implement `AddColumn(b *Board, name string) error` per design TD2 + spec §"AddColumn". Trim name; empty → `*EmptyColumnNameError`; duplicate → `*ColumnAlreadyExistsError`; else append. Done when the helper compiles and a smoke test (`AddColumn` on a 2-column board with a fresh name) appends correctly.
- [ ] 1.3 Implement `RenameColumn(b *Board, from, to string) error` per design TD5 + spec §"RenameColumn". Order: `from == to` no-op; empty `to` → `*EmptyColumnNameError`; unknown `from` → `*ColumnNotFoundError`; duplicate `to` → `*ColumnAlreadyExistsError`; else mutate `b.Board.Columns[idx]` and walk `b.Cards` rewriting every `Column == from` to `Column = trimmed-to`. Do NOT refresh affected cards' `UpdatedAt`. Done when the helper compiles and a smoke test renames `todo` → `backlog` across 3 cards.
- [ ] 1.4 Implement `DeleteColumn(b *Board, name string) error` per design + spec §"DeleteColumn". Order: unknown → `*ColumnNotFoundError`; `len == 1` → `*CannotDeleteLastColumnError`; cards reference → `*ColumnHasCardsError{Cards: [...]}` (collect every card in the column as `affectedCard{ID, Title}`); else delete via `slices.Delete`. Done when the helper compiles and a smoke test deletes an empty column.
- [ ] 1.5 Implement `MoveColumn(b *Board, name string, position int) error` per design TD6 + spec §"MoveColumn". Order: unknown → `*ColumnNotFoundError`; clamp position to `[0, len-1]`; target == current → no-op success; else slice-out + slice-insert. Cards untouched. Done when the helper compiles and a smoke test moves `done` → index 0 in a 3-column board.

## 3. Server routes (`internal/server/handlers.go`)

- [ ] 3.1 In `internal/server/handlers.go`, add four payload struct types: `columnCreatePayload{Name string}`, `columnRenamePayload{Name string}`, `columnMovePayload{Name string; Position int}`. Add the four handlers `handleColumnCreate`, `handleColumnRename`, `handleColumnDelete`, `handleColumnMove` following the V2/V3 pattern (decode → validate → load → mutate → save → respond). Each MUST emit `Content-Type: application/json`. Done when the four handlers compile and `go vet ./internal/server` is clean.
- [ ] 3.2 `handleColumnCreate`: decode body, trim name, call `board.AddColumn`, save, respond `201` with `{"columns": b.Board.Columns}`. Done when `go vet` passes and the handler signature compiles.
- [ ] 3.3 `handleColumnRename`: decode body, trim new name, call `board.RenameColumn(b, r.PathValue("name"), trimmed)`, save, respond `200` with `{"columns": b.Board.Columns, "renamed": {"from": old, "to": trimmed}}`. Done as above.
- [ ] 3.4 `handleColumnDelete`: per design TD4, short-circuit the 404 case before calling `board.DeleteColumn` — if the named column is not in `b.Board.Columns`, emit a 404 envelope inline (`COLUMN_NOT_FOUND`). Otherwise call `DeleteColumn`, save, respond `200` with `{"columns": b.Board.Columns}`. Done as above.
- [ ] 3.5 `handleColumnMove`: decode body, call `board.MoveColumn`, save, respond `200` with `{"columns": b.Board.Columns}`. Done as above.
- [ ] 3.6 In `routes()`, register the four new routes: `POST /api/columns`, `PATCH /api/columns/{name}`, `DELETE /api/columns/{name}`, `POST /api/columns/move`. The trailing `/move` route MUST be registered after the `:name` route OR the router MUST disambiguate; Go's `http.ServeMux` does method-and-path matching since 1.22, so the two POST routes can coexist (one is `POST /api/columns`, the other `POST /api/columns/move`). Done when `go test ./internal/server` builds and the route table is complete.
- [ ] 3.7 In `httpError`, add three new `errors.As` arms for `*board.EmptyColumnNameError` → 400 `INVALID_BODY`, `*board.ColumnAlreadyExistsError` → 400 `COLUMN_ALREADY_EXISTS` (details: `{name}`), `*board.CannotDeleteLastColumnError` → 400 `CANNOT_DELETE_LAST_COLUMN` (details: `{name}`), `*board.ColumnHasCardsError` → 400 `COLUMN_HAS_CARDS` (details: `{column, cards}`). The existing `ColumnNotFoundError` arm (400) stays unchanged for PATCH/move sites; DELETE handles 404 inline per task 3.4. Done when `go vet ./internal/server` is clean.
